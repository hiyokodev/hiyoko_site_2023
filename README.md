# hiyoko_site_2023

## 環境構築
必要なもの
- **VSCode**
ソースコードを書くのに使います
- **Git**
ソースコードのバージョン管理に必要なツールです
- **GitHubアカウント**
Gitによる管理をクラウド上でみんなで行います
- **Node.js**
Vue.jsというフレームワークで開発するのに必要です


## セットアップ

```sh
git clone https://github.com/hiyokodev/hiyoko_site_2023.git
cd hiyoko_site_2023
npm install
```

## 動作テスト

```sh
npm run dev
```
